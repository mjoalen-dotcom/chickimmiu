# D3：IG「下季想看什麼？」Carousel — 文案 + 設計 brief

> **靈感來源**：Matin Kim「IG 留言 → 下季商品 fan-driven loop」模式
> **目標**：把粉絲變成 co-designer；每月 1 次 carousel poll，收 100-300 留言、回覆 3-5 個方向
> **預估互動**：以 chickimmiu 主帳號目前粉絲基數估算，poll 留言 80-200 則為健康
> **發文時機建議**：每月第 1 個週四 19:00-21:00（IG 演算法傾向晚間黃金時段）

## 內容素材庫

### 圖卡可用色票（Korean Cozy Beige）
- 主背景：`#FAF7F2`（奶白）
- 重點色塊：`#E8D5C4`（米杏）/ `#D4A574`（玫瑰金棕）
- 重點文字：`#2C1810`（深褐黑）
- 輔助色：`#FFB5A7`（粉膚）/ `#A0826D`（深米）

### 字體建議
- 中文標題：Noto Serif TC / 思源宋體（粗體）
- 中文內文：思源黑體（中等）
- 數字 / 英文 / 編號：Inter / DM Serif Display
- emoji 用韓系常用：✿ ✦ ⤵ ❤️ ⓘ ✓

### 圖卡製作工具
- **Canva**（最快，有 IG carousel template）
- **Figma**（更彈性，可建可重用元件）
- **Adobe Express**（圖卡 + 動效）

## A/B 測試建議

第一次跑可以同時測：
- **A 版**：純風格選項（10 個選項都是風格 / 單品種類）
- **B 版**：混合選項（5 個風格 + 3 個顏色 + 2 個價位帶）

第一次後檢視：
- 哪一版留言多？哪一版留言「具體」（含 size / 預算 / 場景）？
- 哪些選項被 0 人選 → 下季可砍
- 哪些選項被瘋狂選 → 是否該變主品牌軸線

## 6 個月延伸玩法

如果第一次成功，下面是進階版：
1. **粉絲共創 capsule**：選 1 個選項做成「粉絲共創 capsule」，在商品頁標明「由粉絲投票決定」
2. **預售 unlock 機制**：選項排名 1 → 收 100 件預售自動 -10%、收 200 件 -15%、收 300 件 -20%
3. **入選粉絲 muse**：留言最具洞見的 3 人邀請成「下季 muse」，免費收 1 套 + 拍 lookbook
4. **跨品牌投票**：跟 1-2 個小品牌合作，「這個月你想看哪 3 個品牌進 chickimmiu？」
5. **顏色月**：每月 1 個 pantone 主色，全月圖卡 + 商品都圍繞那個色
6. **產地透明化**：投票完成後做幕後 vlog「我去韓國找這個款的 7 天」，全程 IG live + reel
