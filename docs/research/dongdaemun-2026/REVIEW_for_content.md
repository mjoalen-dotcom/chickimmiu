## TL;DR

韓國 D2C 品牌（Musinsa / ABLY）已經把「AI 內容 stack」commoditize：虛擬模特、Sinsang Studio 一週快拍、AI 文案翻譯、AI 影片生成。chickimmiu 5-10 人團隊只要選對 3 個工具、月費 NT$3-7k 可以替代外拍棚 + 文案外包，**省 NT$30-100k/月**。視覺從現有「甜美 / 日常」轉向「財閥千金感 / DDP 工業質感」是品牌升級槓桿。

## 文案模板（標題公式）

```
[首爾直出 / DDP設計師款] + [核心場景] + [獨特材質/版型] + [品名]
```

範例：
- ✅ `[首爾直出] 聖水洞週末・微光澤抗皺垂墜西裝落地褲`
- ✅ `[DDP設計師款] 弘大畫廊・羊毛混紗復古泡袖針織`
- ❌ `2026 春季新款韓國正韓現貨女裝洋裝` ← 空洞詞彙、SEO 傷

### 信任鉤子（Hook，第一段）

```
本店全品項經由韓國 Sinsang Market 官方品檢直送，100% 東大門 DDP 商圈
原創設計，拒絕粗糙打版。本款由 [批發商名] 設計師親自設計，每週限量
{N} 件。
```

### 材質與身型細節（避開「假韓貨」疑慮）

```
材質：高密度親膚斜紋紗 65% / 萊卡 5% / 羊毛 30%（首爾原產，韓國紡織協會
認證）。立體剪裁、修飾梨形假胯寬。模特身高 168 cm 體重 50 kg、穿 M 號。
```

> 文案模板要存進 chickimmiu admin 的「商品文案 prompt 庫」collection（建議新增）、給 AI 翻譯 / 改寫工具當 system prompt。

## 直播 SOP（IG / TikTok / LINE）

### 直播前 24h
- 預告 reel：30 秒，展示「今晚直播 5 件韓國新品」+ 場景 hook（「DDP 街拍 lookbook」）
- LINE 群（VIP 預售）：先發 lookbook + 商品 ID、開放預訂

### 直播當下（45-60 分鐘）
| 時段 | 內容 | KPI |
| --- | --- | --- |
| 0-5 min | 開場 + 自我介紹 + 今晚 menu（5-7 件） | 即時觀看人數 |
| 5-40 min | 真人試穿 + 即時 Q&A + 限量出清 | conv lift +15%（韓國數據） |
| 40-50 min | 「下次預告」+ LINE 群誘餌 | LINE 群增長率 |
| 50-60 min | 結束 + 重播時段公告 | 直播後 24h 銷售 lift |

### 直播後 48h
- IG carousel post（10 張）：每件商品 1 張試穿 + 1 張平拍
- TikTok 短影片（15-30 秒）：剪集錦 + 高流量片段
- LINE 群：補丟限量提醒 + 完售商品「補貨候補」表單

## 5 個 pitfalls 警告

1. **別用「正韓現貨」「2026 新款」空洞詞彙**：SEO 傷、消費者不信、競品同質
2. **別讓 AI 攝影看起來太假**：背景 / 光線細節要 review、A/B 測 30 天
3. **別跟 Pazzo / Meier.Q 文風相同**：他們偏「日常 / 甜美」，chickimmiu 走「都會 / 質感」
4. **別忽略 LINE 群**：韓國 KakaoTalk、台灣 LINE，VIP 群是高 LTV 槓桿
5. **直播別只賣不互動**：韓國 live commerce 高轉換靠 Q&A 真實感、不是喊價
