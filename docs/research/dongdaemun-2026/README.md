## 5 條最關鍵的可執行 takeaways

從 `Q7-行銷作戰書.md` 萃取（完整版見原檔）：

1. **採購不要打價格戰，押「設計差異化 + 韓國 made-in 故事」**
   廣州 commerce 同款只賣 1/10 價，台灣電商若也走低價必死。轉走「品質 + 故事」差異化（Korean blood 設計師、面料、版型），客單價站穩 NT$1500-3500。

2. **採購主走 Sinsang Market、保留 5-10% 真東大門夜批**
   Sinsang Market 11,000+ 活躍批發商、24,000 筆/天交易、自動品檢、跨境物流一條龍 → 適合 80-90% 起點。剩餘 5-10% 仍直接東大門選品（保留差異化、讓「東大門限定」成為行銷敘事）。

3. **AI stack 起步從這 3 個工具切入（5-10 人團隊）**
   - **AI 商品攝影 / 虛擬模特**（取代外拍棚：每月省 NT$30k-80k；月費 NT$1500-3000）
   - **AI 庫存預測**（demand error 降 20-50%，避免囤貨：每月省 NT$20k+）
   - **AI 文案 / 翻譯**（韓國商品描述自動翻譯 + 在地化：每月省 NT$15k-30k）

4. **行銷檔期跟韓國 SS/FW 同步（不延後）**
   韓國 Seoul Fashion Week（3 月 / 9 月）後 2 週內上架，趁韓劇 / K-pop 爆款熱度。Pazzo / Meier.Q 通常延後 1 個月才跟，搶先就贏。

5. **直播 / IG / TikTok 抄韓國 5 個操作點**
   - 真人試穿 + 即時互動 Q&A（轉換率 +15%）
   - 限量出清（建立稀缺感）
   - 拍攝場景借「DDP-style 高質感工業背景」
   - 預告片 24h、直播後 48h carousel post 重複賣
   - LINE 群預售制（韓國 Naver Talk 同模式）
